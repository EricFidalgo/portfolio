Local Database using a custom Hash Table with Encryption

Local Database Manager with Secure Password Management.

In this project I learned how to create a hash table to not only create a database but to output details very quickly and efficiently.


The program has one main class that deals with the main hash map implementation such as file implementation, rehashing, and encryption for passwords. The second class is the class for the database that will talk to the hash table class by being able to create, add, remove and update user information to the hash table.


Development Period: November 2024
